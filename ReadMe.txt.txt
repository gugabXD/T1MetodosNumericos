Sistema feito para ser bem simples; enviei a pasta "cases" junto para facilitar a execução.

Selecionando a opção 1, é só enviar o nome do arquivo (Exemplo: caso10.txt) que o programa vai procurar na pasta cases e te devolver uma análise mais completa (resultado do método de gauss, Jacobi sem paralelismo e Seidel, tempo de execução dos três).

Selecionando a opção 2, o programa lê todos os arquivos da pasta cases e devolve a solução de cada um de maneira mais enxuta. O arquivo "exemplo.txt" trata do fluxograma dado no próprio pdf do trabalho.
